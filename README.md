# Cmd-Pack

Java Maven项目Git打包工具



## 快速开始

```powershell
java -jar cmd-pack.jar
```



## 配置文件

cmd-pack.yaml

```yaml
basic:
  project:
    name: bxkc
    source: "E:\\Programming\\Code\\Company\\bxkc-pc"
    output: "D:\\Test\\bxkc"
  git:
    author: dunhanson
    after: "2020-04-01 00:00:00"
    before: "2020-04-18 00:00:00"
```
